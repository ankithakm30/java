git init ---it creates a new git repository
git adds a change in the working directory to the staging area
git status ---displays the state of the working directory and the staging area.
git commit -m command --- what you'll use to take all of the changes that have been made locally and push them up to a remote repository.
git log command ---displays committed snapshots.
git branch --- create a new branch, delete branches and rename branches.
git check out --- lets you navigate between the branches created by git branch . 
git config ---set git configuration values on a global or local project level.
git clone ---to point to an existing repo and make a clone or copy of that repo at in a new directory, at another location.
git diff ---comparing changes.
